# **App Name**: WishZep

## Core Features:

- Modern Storefront: Eye-catching design with glassmorphism, mesh gradients, and responsive layout.
- Product Discovery: High-resolution image grids, dynamic routing for SEO, sale badges, and countdown timers.
- Detailed Product Information: Variant selection with visual swatches and sizes, accordion information sections, and trust badges.
- Shopping Cart Management: Realtime add to bag functionality and professional toast notifications using Zustand.
- Secure Admin Dashboard: Role-based access control with auto-redirection for unauthorized users.
- Product Management: Full form to add new products with drag-and-drop image uploads to Cloudinary Storage, inventory control, and dynamic attribute addition.
- User Authentication and Management: Passwordless email login with magic links, automatic profile creation with SQL triggers, and session persistence. Profile image upload to Cloudinary. Firebase for sign in with Google, email, etc.
- Payment Gateway Integration: Integration with Razorpay for secure payment processing.
- Informational Pages: Working pages for Refund Policy, Contact Us, Terms and Conditions, Privacy Policy, Shipping Policy, and About Us.
- Product Search: Search icon and functionality to search products

## Style Guidelines:

- Primary color: Electric Purple (#BE29EC) for a modern, high-energy feel, as if charged with 'Aura.'
- Background color: Very light desaturated purple (#F3E8F7).
- Accent color: Sky Blue (#29A6EC) to complement and contrast the primary, creating a cool, vibrant, modern look.
- Body and headline font: 'Inter', a grotesque sans-serif for a modern, machined look. Its neutrality allows other elements to shine.
- Modern and minimalist icons, consistent with the overall glassmorphism and mesh gradient aesthetic.
- Responsive grid layout optimized for mobile, tablet, and desktop, ensuring a consistent user experience across all devices.
- Subtle, modern animations and transitions to enhance user interaction and provide a polished feel. Rich UI with colorful professional ultra-modern animations effect and glassmorphism 3D
