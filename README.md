## üì¶ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/AdityaChoudhary01/WishZep.git
cd wishzep
```

### 2. Install dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Create a `.env.local` file in the root directory and populate it with your keys (see `.env.example` for the required format):
- Firebase API Keys
- Razorpay Credentials
- Cloudinary Preset
- SMTP/Email credentials

### 4. Run Development Server
```bash
npm run dev
```
