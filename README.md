# WishZep | Premium Modern Aura Shop

WishZep is a high-performance, modern e-commerce platform built for visionaries. It features a curated collection of techwear and performance gear with a high-energy, professional aesthetic.

Visit the live site at: [https://wishzep.shop](https://wishzep.shop)

## ‚ú® Features

- **High-Fidelity Branding**: Custom SVG-based logo system with professional gradients.
- **Advanced Admin Dashboard**:
  - Flipkart-style simplified labels (MRP, Selling Price, Total Stock).
  - Multi-image artifact gallery.
  - Apparel-specific size management and size chart uploads.
  - Comprehensive customer order management with itemized breakdowns and shipping logs.
- **Modern Shopping Experience**:
  - Real-time cart management via Zustand.
  - Native mobile sharing and clipboard integration.
  - Responsive technical gear specification grid.
  - Secure Razorpay payment gateway integration.
- **Authentication Protocol**:
  - Google Sign-in.
  - Email Magic Link.
  - Mobile OTP verification (Firebase Auth).
- **SEO & Performance**:
  - Fully optimized for `https://wishzep.shop`.
  - Dynamic Sitemap and Robots.txt generation.
  - JSON-LD Structured Data for products and organizations.
  - PWA-ready manifest and high-performance image optimization.

## üì¶ Getting Started

### Prerequisites

- Node.js 18+ 
- Firebase Account
- Cloudinary Account (for image hosting)
- Razorpay API Keys

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/wishzep.git
   cd wishzep
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure Environment Variables:
   Create a `.env.local` file and add your credentials.

4. Run the development server:
   ```bash
   npm run dev
   ```
